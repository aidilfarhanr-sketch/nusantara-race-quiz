# Nusantara Race Quiz — Cerdas Cermat Indonesia 1 vs 1

Nusantara Race Quiz adalah game cerdas cermat berbasis web dengan konsep duel 1 vs 1. Game ini mengangkat tema pengetahuan umum Indonesia dengan tampilan modern, interaktif, dan responsive untuk dimainkan di laptop, PC, tablet, maupun HP.

##  Tentang Game

Game ini dibuat untuk menguji pengetahuan umum Indonesia melalui pertandingan dua pemain. Setiap pemain menjawab pertanyaan menggunakan tombol keyboard yang sudah disediakan.

Player yang menjawab benar akan mendapatkan poin dan pelari akan bergerak maju menuju garis finish. Di akhir permainan, sistem akan menampilkan pemenang beserta statistik pertandingan.

---

##  Fitur Utama

- Mode permainan 1 vs 1
- Pilihan level pengetahuan:
  - Mudah
  - Medium
  - Sulit
- Pertanyaan seputar pengetahuan umum Indonesia
- Arena duel merah-biru
- Animasi pelari menuju garis finish
- Timer permainan
- Scoreboard real-time
- Statistik benar, salah, dan total terjawab
- Tombol pause dan resume
- Tombol suara
- Tombol restart
- Halaman hasil akhir pertandingan
- Form kritik dan saran
- Responsive untuk HP, tablet, laptop, dan desktop
